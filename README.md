# D2RSR
**災難救助遙測機器人**

D2RSR的設計初衷是想藉由提供精確的即時環境資訊，來提升救援效率。但我們並不滿足於此，希望能更進一步地提供各方面的援助，因此我們的物件辨識功能、地圖繪製功能應運而生。此外，為了吸引和我們有相同理念的朋友共同參與，我們開源了所有程式碼，希望能達到拋磚引玉的作用。
最後，我們希望能將這套系統推向市場，讓所有人都能看見，現在存在一種不依賴載具、不依賴技術，同時價格低廉的這麼一個系統--D2RSR。
